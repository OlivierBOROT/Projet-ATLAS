# Tests NLP

## Scripts disponibles

### 1. Test complet des modules

```bash
cd NLP/tests
python test_nlp_modules.py
```

Un test/exemple pour voir comment utiliser les 3 modules NLP sur les 2 offres exemples.

### 2. Afficher les lemmes

```bash
cd NLP/tests
python show_lemmas.py
```

Un test/exemple pour voir comment récupérer les lemmes extraits (mots-clés sans stop-words).

## Fichiers

- `example_offers.json` : 2 offres exemples
- `test_result.json` : Résultat du dernier test complet
